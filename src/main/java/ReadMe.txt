Для работы программы сделайте следующие шаги

    1. Создайде папку Вводные на вашем комьютере.
    2. В папке Вводные создайте Excel файл с названием ВводныеСебестоимость.xlsx. В столбце A укажите названия ваших товаров.
В столбце B укажите артикулы продавца. В столбце C укажите себестоимость. В столбце D укажите стоимость фулфилмента на единицу товара,
возможно в вашем случае стоимость упаковок, работы складовщиков и доставку до склада ВБ и т. д.
    3. В папке Вводные создайте Excel файл с названием ВводныеНазванияРК.xlsx. В столбце A укажите названия рекламных кампаний. В столбце B укажите
категории товара, к которым относятся рекламные кампании.
    4. Создайте пустую папку с названием Отчеты0807_1407, где цифры обозначают даты создаваемого отчета.
    5. В файле ExcelConverter (его нужно выбрать на панеле слева) пропишите путь к папке, созданной в пункте 4, в формате
"e:\\Proga\\AnaliticsWB\\Отчеты0807_1407\\" вместо прописанного там путя на данный момент примерно в 25-ой строке
    6. С Wildberries скачайте Excel отчет с детализацией, сохраните его в папку, созданную в пункте 4, и назовите ОтчетДетализация.xlsx
    7. В кабинете Wildberries зайдите в Аналитика - Отчеты - Платное хранение. Выберите вкладку Отчет по номенклатурам.
Выберите период такой же как у вас в детализации. Нажмите Выгрузить Excel. Назовите файл ОтчетХранение.xlsx. Сохраните в папку,
созданную в пунтке 4.
    8. В кабинете Wildberries зайдите в Реклама - ВБ.Продвижение. Зайдите во вкладку Финансы. Выберите период такой же как у вас
в детализации и захватите в это период еще 2 предыдущие неделю. То есть, если вы делаете отчет за неделю с 24.06.24 по 30.06.24, то нужно
выбрать даты с 10.06.24 по 30.06.24. Нажмите скачать Excel. Укажите для скачивания папку. Назовите файл ОтчетРеклама.xlsx
    10. Перейдите слева на панели в файл ExcelConverter нажмите сверху зеленую стрелочку для запуска. Слева от зеленой стрелочки должно
быть выбрано Current File
    11. В папке появится файл UnitЭкономика.xlsx. Откройте его
